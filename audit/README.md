# Audit Workspace

Este diretório é a área canônica para inventário, checklist, matriz de rastreabilidade, dashboard e relatório final de governança.

Os espelhos derivados de arquivo a arquivo não ficam mais aqui. Eles foram segregados para `artifacts/audit/files_analysis/`, porque servem como material analítico e não como fonte canônica da auditoria.

## Artefatos obrigatórios

- `audit/master_governance_checklist.md`
- `audit/traceability_matrix.md`
- `audit/validation_log.md`
- `audit/governance_dashboard.html`
- `audit/final_governance_report.md`
