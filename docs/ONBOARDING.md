# Guia de Onboarding (ONBOARDING.md)

Bem-vindo(a) ao BirthHub360!

1. Clone o repositório.
2. Siga o arquivo `docs/engineering/configuration-onboarding.md` para configurar os `.env`.
3. Certifique-se de ter `Node.js 20+`, `pnpm 9+`, `Python 3.12+` e Docker instalados.
4. Execute `pnpm install`.
5. Execute `docker compose up -d` para subir banco e serviços locais.
6. Execute `pnpm dev` para subir os apps da stack.
7. Para contribuir, leia nosso `CONTRIBUTING.md`.
