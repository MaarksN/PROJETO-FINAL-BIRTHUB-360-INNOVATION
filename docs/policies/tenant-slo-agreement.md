# SLO por Tenant (Service Level Objective)

- **Disponibilidade (Availability):** 99.9% / mês (aproximadamente 43 minutos de downtime aceitável/mês).
- **Latência (P95 Garantida):** <= 400ms para todos os requests não intensivos (como requisições CRUD nativas) e SLA de resposta em até 10s para execuções sincrônicas complexas de Agentes via webhooks long-polling.
