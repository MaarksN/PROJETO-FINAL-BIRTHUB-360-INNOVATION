# Análise de Custo e Validação de Preços

O custo unitário estimado por mil tokens na nuvem LLM escolhida, somado à proporção por MB no banco RDS, orienta os preços dos planos base (Starter e Pro) garantindo uma Margem Bruta Alvo (Target Gross Margin) de >75%.
Os limites rígidos impostos em cada tier impedem que abusos de um cliente (noisy neighbor) erodam a rentabilidade geral da plataforma.
