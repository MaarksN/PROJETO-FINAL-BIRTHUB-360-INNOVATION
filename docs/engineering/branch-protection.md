# Política de Branch Protection

- Branch `main` protegida contra pushes diretos.
- Requer Pull Request.
- Mínimo de 1 aprovação de reviewer.
- Code Owners devem aprovar mudanças nos seus respectivos diretórios.
- Todos os status checks (CI) devem passar.
- Branches devem estar atualizadas com a `main` antes do merge.
