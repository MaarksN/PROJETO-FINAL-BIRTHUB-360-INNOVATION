# Checklist de Aceite do CI

O que significa 'Pipeline Verde':
1. Checkout do código bem-sucedido.
2. Instalação de dependências sem erros.
3. Linting sem erros e sem warnings não catalogados.
4. Type checking TypeScript completado com sucesso.
5. Suíte de testes (Python e Node) executada e aprovada.
6. Build de todos os apps concluído.
