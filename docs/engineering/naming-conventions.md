# Naming Conventions

## Arquivos e Pastas
- `kebab-case` para pastas e nomes de arquivos de componentes/módulos principais (ex: `user-profile.tsx`, `auth-service.ts`).

## Variáveis e Funções
- `camelCase` para instâncias, variáveis e funções.
- `PascalCase` para Classes, Tipos e Interfaces.
- `UPPER_SNAKE_CASE` para constantes globais.
